understanding group_by

The group_by method is useful if you want to group elements based on a particular criterion. The method takes an array or a range and returns a hash. The keys of the new hash are determined by the code block, and the values of the hash are arrays of the elements from the original array that correspond to each key. An example or two will help.

Let's say you have an array of strings, and you want to group them by length:

words = ["Ball", "Bat", "Hoop", "Marbles", "Hat", "Bear"]
words.group_by{|word| word.length}

=> {7=>["Marbles"], 3=>["Bat", "Hat"], 4=>["Ball", "Hoop", "Bear"]}


We can take the same array and group the elements by their first letter:

words = ["Ball", "Bat", "Hoop", "Marbles", "Hat", "Bear"]
words.group_by{|word| word[0]}

=> {77=>["Marbles"], 66=>["Ball", "Bat", "Bear"], 72=>["Hoop", "Hat"]}

In this case, the keys to the hash are the ascii code for the first letter.
